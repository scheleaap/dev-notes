## Google Meet

Create file `~/.local/share/applications/google-meet-qrn-imsx-abi.desktop`:

```
[Desktop Entry]
Type=Application
Encoding=UTF-8
Name=Team
Exec=chromium-browser --app=https://meet.google.com/aaa-bbbb-ccc --new-window
Icon=google-meet.png
Terminal=false
```

Store the icon in `~/.local/share/icons`.
